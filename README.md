# Traffic Accident Analysis Dashboard

## Overview
The Traffic Accident Analysis Dashboard is an interactive tool designed to analyze and visualize the Traffic Accident Reports in Abu Dhabi Dataset 2025. This dashboard helps stakeholders gain actionable insights into traffic accident patterns, supporting public safety initiatives, urban planning, and transportation policies.

## Features
- **Interactive Filters:** Filter data by city, time, weather conditions, and accident types.
- **Dynamic Visualizations:** View accident locations on a map and analyze accident causes through charts.
- **Actionable Insights:** Identify high-risk areas and times for targeted safety measures.

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/traffic-accident-dashboard.git
   
2. Navigate to the project directory:
   bash
   cd traffic-accident-dashboard
   
3. Install required dependencies:
   bash
   pip install -r requirements.txt
   

## Usage
1. Ensure you have the dataset available at the specified URL.
2. Run the application:
   bash
   python app.py
   
3. Open your web browser and navigate to `http://127.0.0.1:8050/`.

## Dataset
The dataset includes key variables such as:
- **Time and Location:** Date, time, city, and street of the accident.
- **Environmental Conditions:** Weather and road surface conditions.
- **Accident Details:** Type of accident, cause, and number of injuries.
